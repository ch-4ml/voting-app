# voting-app
교수님께서 의탁하신 교회 투표 앱입니다.

# 개선 중
- 임시로 선거권자 status를 강제로 2(권사 선거)로 설정해놓음
- 선거 생성 시 선거권자 / 피선거권자 목록 Insert 하는 동안 프로그레스 바 표시
- 인증번호가 로그에 표시되는 현상
- 선택한 피선거권자 칸 색상 변경
- 생년월일 형식 수정
- 이름이나 숫자로도 투표할 수 있도록 수정(검색)
- 투표한 사람 수 세기 (총 투표한 수, 사람 수 둘 다 필요)
- 후보자 정보를 제공할 수 있는 다른 방법 필요
- 관리자 - 특정 후보자의 투표 기록을 초기화 할 수 있는 방법 필요
- 선거 종류를 선택해서 선거할 수 있게 변경
- 총 득표수 0인 경우 예외 처리
